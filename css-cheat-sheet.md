## display
> display among other elements or/and in the line
* inline
  > [element][element]
  > min height & width - no control 
* block
  > \[ ......element....... \]
  > fill the whole row on the screen
  > height & width
* inline-block
  > [element]
  > with height & width
* flex
  > [element][element][element]
* table 
  > [element][element][element]
  > with predefined hight
* grid
  > [element]\n 
  > [element]\n
* none   
  > no element in layout, no dedicated place
